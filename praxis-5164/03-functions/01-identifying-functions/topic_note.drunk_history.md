A mathematical relation is nothing more than a set of ordered pairs mapping an input value to an output value. It’s just a historical record of what happened! Input: Jimmy. Output: getting stuffed in a locker. It's anarchy! But we are educators. We demand structure. We are looking for a function, which is a specific type of mathematical relation where each input value maps to exactly one output value. One kid, one assigned seat on the bus. Period. 

To govern this system, you need to know your boundaries. The domain of a function is the complete set of all possible independent input values—that is the absolute, finalized roster of every feral pre-teen you are legally allowed to load onto that bus. Meanwhile, the range of a function is the complete set of all possible dependent output values—the actual list of destinations where those kids can end up. 

If you are auditing this system, you have to look for the fakes, the glitches in the matrix. If you're staring at a clipboard, a set of ordered pairs represents a function if no two ordered pairs share the same first element with a different second element. If your clipboard says Jimmy goes to seat 4, and another line says Jimmy goes to seat 12, Jimmy has been physically cloned and you have a massive problem. It’s the same when you’re scanning the school’s digital spreadsheet. A table of values represents a function if no two distinct rows contain the same input value paired with different output values. Two different kids can absolutely go to the same seat—that's just a buddy system. But one kid mapped to two different realities? The table is a lie!

The kids will try to hide this chaos in algebra, usually by trapping the output variable in a reality TV drama. You have to be incredibly suspicious of exponents here, because an algebraic equation containing an output variable raised to an even power generally does not represent a function. Imagine setting up the equation $y^2 = x$. If you feed it an innocent input of $x = 16$, that output $y$ dramatically splits its personality into two evil twins: $4$ and $-4$. One input just birthed two outputs! You also have to watch out for the mathematical witness protection program: an algebraic equation containing the absolute value of the output variable generally does not represent a function. Slap absolute value bars around your output, like $|y| = x$, and an input of $7$ instantly fractures into an output of $7$ and $-7$. It’s uncontrollable!

When you map these kids out on a coordinate plane, spotting the chaos becomes an action movie. The vertical line test is a visual method used to determine if a graphical relation is a function. Picture the laser security grid in a museum heist dropping straight down from the ceiling. A graph represents a function if no vertical line drawn on the coordinate plane intersects the graph at more than one point. If that vertical laser slices through a curve twice, BEEP BEEP BEEP! One $x$-input is existing in two $y$-locations at once! Lock down the building!

Once you have established total functional control, you use the official stamp of authority. In function notation, the symbol $f(x)$ represents the output value of the function $f$ for the specific input value $x$. It isn’t "$f$ times $x$," it is the sealed, stamped destiny of kid $x$ after going through the $f$ machine. Now you have to actually run the machine, which we call evaluating. Evaluating a function algebraically involves substituting a specific domain value for the independent variable in the function equation. You literally shove the kid's specific value into the algebraic woodchipper of the equation and catch the number that spits out. If you are doing this visually, evaluating a function from a graph requires finding the specified input value on the horizontal axis and identifying the corresponding output value on the vertical axis. It’s like playing Battleship! Walk down the horizontal hallway to find the kid, take the elevator straight up to the line, and stare directly across at the vertical wall to read their fate.

But beware the cafeteria sorting hat: piecewise functions. Evaluating a piecewise function requires identifying which domain interval contains the specified input value before substituting the input into the corresponding expression. If you have a 6th grader, you can't just shove them into the 8th-grade equation! You have to check the domain interval signs, figure out which turf they belong to, and route them to their one specific, legally assigned algebraic expression.

Once you know the rules, you will realize your classroom is dominated by three main gangs. First, you have the marching band: linear functions. A linear function is represented algebraically by an equation of the form $f(x) = mx + b$. There are no divas here, because the highest power of the independent variable in a linear function equation is exactly one. Because they are so relentlessly steady, the graph of a linear function is a straight line. Unflinching! To spot them hiding in a table, you measure their footsteps by checking the first difference of a mathematical relationship, which is the difference between consecutive output values for equal intervals of input values. A table of values represents a linear function if the first differences are a non-zero constant. Step by step, it’s always the exact same size gap.

Next, you have the drama queens. A quadratic function is represented algebraically by a polynomial equation of degree two. They are powerful, but there is a strict uniform code: the standard form of a quadratic function is $f(x) = ax^2 + bx + c$. And listen to me, in the standard form of a quadratic function $f(x) = ax^2 + bx + c$, the coefficient $a$ must be a non-zero real number! If that $a$ is zero, the $x^2$ vanishes, their drama is revoked, and they are demoted back to a boring linear straight line. Because their moods swing with such wild acceleration, the graph of a quadratic function is a U-shaped curve known as a parabola. You can’t just look at their surface-level footsteps in a table; you have to dig into their psyche. You must look for the second difference of a mathematical relationship, which is the difference between consecutive first differences. A table of values represents a quadratic function if the second differences are a non-zero constant for equal intervals of input values. It’s the hidden, pulsing heartbeat beneath their chaotic surface!

Finally, heaven help us, you have the viral TikTok trends. An exponential function is represented algebraically by an equation where the independent variable appears as an exponent. The kids have literally climbed into the algebraic rafters! The standard form of an exponential function is $f(x) = ab^x$. But the authorities heavily regulate virality. In the exponential function equation $f(x) = ab^x$, the base $b$ must be a strictly positive constant! If that base were negative, the moment some kid tries a fractional exponent, you'd be taking the square root of a negative number and you would rip a hole straight out of the real number universe! Furthermore, in the exponential function equation $f(x) = ab^x$, the base $b$ cannot equal one. Multiplying by $1$ forever is flat, pathetic, and frankly, a mathematical tragedy. Visually, these trends explode upward, but the graph of an exponential function features a horizontal asymptote—an invisible, agonizing glass ceiling or floor that the curve gets infinitely close to but can absolutely never touch. Because they grow by multiplying, not adding, you must abandon differences entirely when auditing their tables. A table of values represents an exponential function if the ratio of consecutive output values is a constant factor for equal intervals of input values. You divide the outputs! And that constant ratio of consecutive outputs in an exponential function table represents the base multiplier of the exponential model. That ratio *is* the $b$!
