Because of bad prep work. Specifically, corrupted temporary cache files can cause a mobile application to freeze in place like a deer in headlights. The first step in your troubleshooting workflow? Sweep the station into the trash! Clearing a mobile application's cache removes temporary operational files, forcing the app to download a fresh, clean set of ingredients without logging the user out.
