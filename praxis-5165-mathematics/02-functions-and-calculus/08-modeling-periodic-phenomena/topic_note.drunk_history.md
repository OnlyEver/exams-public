While the period tells you the length of time it takes to finish one cycle, frequency represents the number of complete cycles a periodic function undergoes per unit of the independent variable. If the period of a trend is a quarter of a semester, the frequency is four trends per semester!

To find the time $x$, you have to work backward using inverse trigonometric functions. But here is the crisis! Sine, cosine, and tangent are chronic repeat offenders; they fail the horizontal line test infinitely many times. To interrogate them, we have to put them in mathematical witness protection. Inverse trigonometric functions require restricted domains of the parent trigonometric functions to satisfy the definition of a true mathematical function, guaranteeing one input gives exactly one output. 

We lock them in solitary confinement. The restricted domain of the function $y = \sin(x)$ used to create the inverse sine function is precisely $[-\pi/2, \pi/2]$. Consequently, the range of the inverse sine function $y = \arcsin(x)$ is exactly $[-\pi/2, \pi/2]$. You need to know that inverse trigonometric functions output an *angle* measure corresponding to a given trigonometric ratio. So, the expression $\arcsin(x)$ literally represents the specific angle $\theta$ locked in the interval $[-\pi/2, \pi/2]$ such that $\sin(\theta) = x$. 

Cosine's cell block is different. The restricted domain of the function $y = \cos(x)$ used to create the inverse cosine function is $[0, \pi]$, meaning the range of the inverse cosine function $y = \arccos(x)$ is strictly $[0, \pi]$. The restricted domain of the function $y = \tan(x)$ used to create the inverse tangent function is $(-\pi/2, \pi/2)$, and the range of the inverse tangent function $y = \arctan(x)$ is $(-\pi/2, \pi/2)$. Notice the parentheses! No brackets! Tangent gets no locked doors because of its vertical asymptotes!
